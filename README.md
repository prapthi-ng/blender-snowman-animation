# blender-snowman-animation

This mini project demonstrates basic animation principles using Blender:
- Object parenting
- Rotation and jump animation
- Intentional timing (90° rotations)
- Snowfall effect
- Text appearance (“Happy New Year”)

## About the Project

This animation was created as part of my learning journey toward Pipeline Development and Technical Animation, focusing on understanding how animated elements move, interact, and are managed inside the software.


Animation preview is provided via screen recording.
